# bd2wg

将 Bestdori 故事导入 WebGAL, 并**下载**相关资源.

## 特性

- **便捷使用**: 选择故事脚本和导出路径, 自动执行导入.

- **脚本转译**: 支持绝大多数 Bestdori 脚本指令.

- **资源下载**: 自动解析并下载相关资源, 告别繁琐的人工爬取.
